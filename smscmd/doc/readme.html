smscmd.jar

Description
This bundle provides a front-end command (for OSCAR shell) to smssender.jar bundle that send SMS (Short Message Service) messages
For more details on how to use this bundle, refer to the /doc/readme.html file embedded in the bundle jarfile and to smssender.jar embedded documentation.
